[1] MLOps · End-to-End ML System

FinPredict — Intelligent Loan Approval System

XGBoost · FastAPI · React.js · MLflow · Apache Airflow · DVC · Docker · Prometheus · Grafana Abstract

FinPredict tackles the problem of manual, inconsistent loan approval decisions in financial institutions. An XGBoost classifier trained on 52,000 loan records predicts approval or rejection from 24 applicant features, served through a FastAPI backend and a React frontend. The full MLOps lifecycle is covered: DVC for two-version data tracking, MLflow for experiment tracking and model registry, a 7-task Airflow DAG for pipeline orchestration, and Prometheus + Grafana for real-time monitoring and drift detection — all containerized with Docker Compose.

[2] Full-Stack · MERN

KIIT International Student Portal

HTML5 · CSS3 · JavaScript · React.js · JWT · Node.js · Express.js · Firebase · MongoDB Abstract

A platform designed to streamline the visa renewal process for international students at KIIT University, facilitating communication and secure document exchange between students and the International Relations Office (IRO) to prevent visa expiry fines. Implemented Firebase Authentication for user management, a Node.js/Express.js backend for secure data storage and communication with MongoDB, and digital document upload/management using Multer middleware. Access control is secured with JWT-based authentication, and the system exposes RESTful APIs for efficient data exchange between frontend and database.

[3] Browser Automation

AMT — Task Automation Extension

JavaScript · Chrome Extension APIs · HTML Abstract

A browser extension built to automate task selection and acceptance for data-entry jobs on microworkers.com. The extension removes the manual overhead of searching and accepting tasks, letting users act on available jobs faster and focus their time on completing work rather than hunting for it — built to run seamlessly across both desktop and mobile browsers.
